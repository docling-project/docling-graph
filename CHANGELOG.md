## v0.2.0 (2026-01-21)

### Features
- Initial release with VLM and LLM-based knowledge graph extraction
- Support for multiple LLM providers (OpenAI, Mistral, Gemini, WatsonX, Ollama, vLLM)
- Flexible extraction strategies (one-to-one, many-to-one)
- Graph export formats (CSV, Cypher, JSON, Docling)
- Interactive HTML visualization
- CLI interface with configuration wizard
- Comprehensive test coverage

### Documentation
- Added setup guides and examples
- Created Pydantic template documentation
- Added WatsonX integration guide
